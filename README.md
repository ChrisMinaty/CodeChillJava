# Code Chill Java – Code aus den YouTube-Videos

Dieses Repository enthält alle Code-Beispiele aus meinen YouTube-Videos. Jedes Snippet zeigt einen typischen Fehler, der Einsteigern aber auch Profis passiert – und wie man ihn versteht und behebt.

## 🎯 Ziel

Das Repository dient als **Begleitmaterial** zu meinen Videos.  
Der Code ist bewusst **einfach**, **kurz** und **anfängerfreundlich** gehalten.  
Er soll helfen, typische Fehler zu erkennen, zu verstehen und zu beheben.

## 🧑‍🏫 Für wen ist das gedacht?

- Java-Entwickler, die mit nervigen Bugs zu kämpfen haben   
- alle, die wie ich hinter die Ursache von typischen Fehler kommen wollen  

## ⚠️ Hinweis

Der Code ist **nicht** für den produktiven Einsatz gedacht, sondern ausschließlich zum Lernen, Ausprobieren, Nachvollziehen.
